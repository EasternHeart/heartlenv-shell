Getting Started

Launching the Xfce Panel

The panel will useally be started as part of your Xfce session when you start the Xfce desktop enviroment by using the startheartlenv script or a login manager. If you want to start the panel by hand, you can run the heartlenv-panel command.

Migration dialog

Migration dialog

When the panel is started for the first time it will give you the following three options:

Migrate old config

This option will only be available when an old 4.6 panel configuration is found on your system. It will try to migrate the old panel layout to the new panel configuration in Xfconf.

Use default layout

This will import the default panel configuration with most of the default panel plugins. If you use the Xfce Panel for the first time, this will be a good base to create your own panel configuration.

Default panel layout
One empty panel

This will give you one empty panel window. Might be what you want if you have already used Xfce Panel before and you know what you want.

Once you have selected an option, your panel will appear. Read more about changing the configuration of the panel and the various plugins in the other sections of this manual.
